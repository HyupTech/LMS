
# 프로젝트 소개 - LMS

인프런 워밍업 클럽 2기 백엔드 코스에서 진행되는 프로젝트입니다.  
프로젝트는 도서관리시스템(Library Management System) 입니다.

## 기능소개
**기본기능**
- 도서 추가 : 도서를 데이터베이스에 추가
- 도서 수정 : 도서 정보 수정
- 도서 삭제 : 도서 삭제 기능
- 도서 조회 : 전체 도서 목록 조회 & 특정 도서 목록 조회

**대여 관리 기능**

- 대여 기록 추가
- 대여 기록 수정
- 대여 기록 삭제
- 반납 기록

## Dependencies

- Spring Web
- Thymeleaf
- Spring Data JPA
- mysql driver
- h2 database
- validation
- Spring Security(추후 추가 예정)
